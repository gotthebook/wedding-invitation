# 💍 Wedding Invitation
 
A mobile-first web-based wedding invitation built while learning HTML, CSS, and JavaScript.
 
## 🌟 Features
 
- Mobile-first responsive design
- Countdown timer to the wedding day
- Venue information with Google Maps integration
- Guestbook / wishes section
- Monetary gift information

## 🛠️ Built With
 
- **HTML** — page structure
- **CSS** — custom styling
- **JavaScript** — interactivity and dynamic content
- **[Vite](https://vitejs.dev/)** — build tool and local dev server
- **[Tailwind CSS v4](https://tailwindcss.com/)** — utility-first CSS framework

## 📱 Design Focus
 
This project is primarily designed for **mobile screens**, reflecting how most guests will open a wedding invitation — on their phone.
 
## 🎓 Learning Notes
 
This project was built as a hands-on learning experience. Key concepts practiced:

- Utility-first CSS with Tailwind
- DOM manipulation with vanilla JavaScript
- Working with a modern build tool (Vite)
